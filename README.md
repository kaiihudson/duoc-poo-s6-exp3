## Descripción general del sistema.
> Este proyecto corresponde a la semana 6 de clases, en este proyecto se revisa la capacidad de Java de extender clases, generando un proyecto, en el cual se trabaja esta capacidad para mostrar información referente a Unidades Operativas
---
## Estructura de Carpetas
```
├──src
   └───main
       └───java
           └───org
               └───salmonttapp
                   ├───data // includes data handlers
                   ├───model // includes models
                   └───ui // includes main invocation
```
---
## HOW TO COMPILE / RUN

1. Clonar este repositorio.
2. Abrir este proyecto en un IDE java (JAVA 25)
3. Ejecutar el archivo `Main.java` desde el paquete `ui`
